Que es este projecto?
Este proyecto es una recreacion de 2 paginas: Una de Home y otra de Contacto
Ambas paginas son totalmente responsive, todas las imagenes contienen descripciones alt, etc.

Que modalidades DOM contiene?
Las modalidades DOM que presenta son las siguientes:
-Validacion de Formularios (Ambas paginas)
-Cambio de imagen/texto con botones (Solo home)
-Carrousel de Imagenes (Solo home)

Como inicio el proyecto?
Basta con abrir el index html
(El index html es la misma pagina que el Home html, ha sido clonada para mas facilidad de entender)
Podras acceder a ambas paginas utilizando los botones de Home y Contact del nav de la parte mas superior de cada pagina.
(El resto de botones no funcionan debido a que no existen las demas paginas)
